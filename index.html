I haven’t quite decided what to write about though. While I’m thinking about what to write I can show some features of asciidoc. You can have links; for example, here is one of my favourite websites.

Paragraphs are easy to do. Just make a new empty line. Here you could write some code. If you want to quote someone, you can do it this way:

How many roads must a man walk down[?]

— Bob Dylan

See that up there? That’s a line.

If you want to use footnotes, here’s how you do it.[1]

Do you have a few questions, and maybe answers? Try the Q&A mode:

Frequently asked questions

  1. Do you like me?

    No.

  2. What’s wrong with cats?

    They’re inherently evil.

  3. What’s the matter?

    Does it matter?

Well! What about that?
